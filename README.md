# Nitro-Dotfiles
Mis dotfiles, no pretendo que sea nada impresionante, solo son para dejar Hyprland con Waybar medianamente bonito y minimal.


## Paquetes necesarios

### Jetbrains Nerd Font:

- Instalación: sudo pacman -S ttf-jetbrains-mono-nerd
- Para actualizar las fuentes: fc-cache -fv

### Wlogout:

- Instalación: paru -S wlogout

### Swaybg:

- Instalación: sudo pacman -S swaybg

### Rofi:

- Instalación: sudo pacman -S rofi

### NetworkManager Menú:

- Instalación: sudo pacman -S networkmanager-dmenu
